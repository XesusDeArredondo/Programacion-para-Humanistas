# Programación para Humanistas

## Español

Este repositorio recopila el código desarrollado durante las investigaciones y ejercicios publicados en mi Substack **[DES]AMBIGUACIÓN**.

Su objetivo es documentar mi proceso de aprendizaje en programación y mostrar cómo herramientas como **Python** y **QGIS** pueden aplicarse al análisis de datos en ciencias sociales, historia, política, filosofía y medio ambiente.

Aquí encontrarás proyectos que evolucionan desde ejercicios básicos hasta análisis cada vez más complejos, reflejando el crecimiento del repositorio y de mis habilidades como programador.

Puedes leer las publicaciones relacionadas en:
👉 https://substack.com/@desambiguacion

## English

This repository contains the code developed for the research and tutorials published in my Substack **[DES]AMBIGUACIÓN**.

Its purpose is to document my learning journey in programming and to demonstrate how tools such as **Python** and **QGIS** can be applied to data analysis in the social sciences, history, politics, and environmental studies.

The projects evolve from simple exercises to increasingly sophisticated analyses, reflecting both the development of the repository and my growth as a programmer.
